# Приложение "Телефонный справочник"

Это простое приложение телефонного справочника, реализованное с использованием архитектурного паттерна Model-View-Controller (MVC). Оно позволяет пользователям управлять списком контактов, включая добавление, удаление и изменение записей.

## Структура

Приложение разделено на следующие модули:

-   `model.py`: Содержит классы данных (`Phonebook`, `Contact`) и класс для работы с файлами (`FileHandler`), а также пользовательские классы исключений. Этот модуль управляет хранением и обработкой данных.
-   `controler.py`: Включает класс `PhonebookController`, который управляет взаимодействием между моделью и представлением. Он извлекает данные из модели и передает их в представление для отображения.
-   `view.py`: Содержит класс `PhonebookView`, отвечающий за отображение данных и взаимодействие с пользователем. Он получает данные от контроллера и отображает их пользователю.
-   `main.py`: Точка входа в приложение. Он создает экземпляры контроллера и представления и запускает приложение.
-   `phonebook.txt`: Этот файл (создается, если его нет) хранит данные телефонного справочника. Каждая строка представляет собой контакт в формате: `Имя, Телефон, ID:ИдентификаторКонтатка`.

## Требования

-   Python 3.6 или выше
-   Не требуются внешние библиотеки (только модули стандартной библиотеки).

## Установка и настройка

1.  Загрузите или клонируйте репозиторий на свой локальный компьютер.
2.  Перейдите в каталог проекта.
3.  Убедитесь, что Python установлен и доступен в вашей среде.

## Использование

1.  Откройте терминал или командную строку.
2.  Перейдите в каталог, содержащий файл `main.py`.
3.  Запустите приложение, используя следующую команду:

    ```bash
    python main.py
    ```

4.  Приложение отобразит меню со следующими опциями:

    -   Заменить Alan на Alya
    -   Удалить Alex
    -   Показать содержимое
    -   Выход

5.  Введите номер, соответствующий желаемому действию, и нажмите Enter.

### Пример сценария

1.  Запустите приложение: `python main.py`
2.  Выберите отображение содержимого телефонной книги, введя `3`. Будет показано исходное содержимое (Alan, Alex, Alice, Adelina), если файл `phonebook.txt` изначально не существовал и был создан приложением.
3.  Выберите замену Alan на Alya, введя `1`.
4.  Выберите удаление Alex, введя `2`.
5.  Снова выберите отображение содержимого, введя `3`. На этот раз Alan будет заменен, а Alex удалена.
6.  Выйдите из программы, введя `4`.

## Функциональность

-   **Заменить Alan на Alya:** Заменяет контакт Alan на Alya в телефонном справочнике. Если Alan не найден, будет отображено сообщение.
-   **Удалить Alex:** Удаляет контакт Alex из телефонного справочника. Если Alex не найдена, будет отображено сообщение.
-   **Показать содержимое:** Отображает содержимое телефонного справочника, показывая каждый контакт на новой строке.
-   **Выход:** Выходит из программы.

## Хранение данных

Данные телефонного справочника хранятся в файле с именем `phonebook.txt`. Если файл не существует, приложение создает его и добавляет начальные данные. Файл обновляется после каждого изменения (замены или удаления), чтобы сохранить изменения.

Формат каждой строки в файле: `Имя, Телефон, ID:ИдентификаторКонтатка`. Например:

Alan, +7 903, ID:907!
Alex, +7 345, ID:907.
Alice, +7 346, ID:9800.
Adelina, +7 367, ID:9678.

## Паттерн MVC

Это приложение следует паттерну MVC для разделения ответственности и повышения удобства сопровождения:

-   **Модель**: Модуль `model.py` управляет данными и бизнес-логикой. Классы, такие как `Phonebook`, `Contact` и `FileHandler`, являются частью Модели.
-   **Представление**: Модуль `view.py` обрабатывает представление данных и взаимодействие с пользователем. Класс `PhonebookView` является частью Представления.
-   **Контроллер**: Модуль `controller.py` действует как посредник между Моделью и Представлением. Класс `PhonebookController` является частью Контроллера. Он обрабатывает ввод пользователя и соответствующим образом обновляет Модель и Представление.

Такое разделение делает код более модульным, простым в тестировании и удобным в сопровождении.
